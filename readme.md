# Last elements of Array using typesccript 

> get last elements from array using typescript. 

## Install

```sh
npm install @leonetti/array-last-ts
```

## Usage

```js
import {last} from '@leonetti/array-last-ts';

last(['a', 'b', 'c', 'd', 'e', 'f']);
//=> 'f'


last(['a', 'b', 'c', 'd', 'e', 'f'], 3);
//=> ['d', 'e', 'f']

```
